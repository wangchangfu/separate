Interpolar Pontos

A ferramenta Interpolar Pontos permite a você prever os valores em novos locais baseado em medidas de uma coleção de pontos. A ferramenta obtém dados do ponto com valores em cada ponto e retorna áreas classificadas por valores preditos. Por exemplo

  • Uma área de gerenciamento da qualidade do ar tem sensores que medem níveis de poluição. Interpolar Pontos pode ser utilizada para predizer níveis de poluição em locais que não têm sensores, como locais com populações de risco—escolas ou hospitais, por exemplo.
  • Prediga concentrações de metal pesado em colheitas baseado em amostras tiradas de plantas individuais.
  • Prediga níveis de nutriente do solo (nitrogênio, fósforo, potássio, e assim por diante) e outros indicadores (como condutividade elétrica) para estudar suas relações com rendimento da cultura e prescrever quantidades precisas de fertilizante para cada local no campo.
  • Os aplicativos meteorológicos incluem predição de temperaturas, chuva e variáveis associadas (como chuva ácida).

Escolher campo para interpolar

Escolha o campo contendo os valores que você deseja interpolar.

Otimizar para

Escolha sua preferência para velocidade versus precisão. Predições mais precisas tomam mais tempo para calcular.

Erros de predição de saída

Se selecionada, uma camada de erros padrão para as predições de interpolação será criada. O nome da camada de saída será o nome da camada resultante com Erros anexados.

Erros padrão são úteis, pois fornecem informações sobre a confiabilidade dos valores preditos. Uma regra geral simples é que o valor verdadeiro cairá dentro de dois erros padrão de 95 por cento de tempo do valor predito. Por exemplo, suponha que um novo local obtém um valor predito de 50 com um erro padrão de 5. Isto significa que a melhor suposição da ferramenta é que o valor verdadeiro neste local seja 50, mas pode ser razoavelmente tão baixo quanto 40 ou tão alto quanto 60. Para calcular este intervalo de valores razoáveis, multiplique o erro padrão por 2, adicione este valor ao valor predito para obter o máximo do intervalo, e subtrai do valor predito para obter o mínimo do intervalo.

Classificar por

Escolha como valores preditos serão classificados em áreas.

  • Intervalo geométrico—As áreas são baseadas em intervalos de classe que têm uma série geométrica. Este método garante que cada intervalo de classe tenha aproximadamente o mesmo número de valores dentro de cada classe e que a alteração entre intervalos seja consistente.
  • Intervalo idêntico—Áreas são criadas de foma que os valores preditos do intervalo sejam iguais para cada área.
  • Área idêntica—As áreas são criadas de forma que os valores do número de dados em cada área sejam iguais. Por exemplo, se os dados tiverem mais valores grandes do que valores pequenos, mais áreas serão criadas para valores grandes.
  • Manual—Esta opção permite a você definir seu próprio intervalo de valores para áreas. Estes valores serão inseridos no parâmetro Valores da quebra de classe abaixo.

Número de classes

Especifique o número de classes (intervalo de valores preditos) na camada resultante. Cada classe define o limite das áreas. Os valores destas classes serão determinados pelo parâmetro Classificar por acima.

Valores da quebra de classe

Insira os valores da quebra de classe desejados separados por espaços. Por exemplo, se você desejar quebras de classe em valores de 1.5, 3, e 7, insira 1.5 3 7. Estes valores definirão o limite superior de cada classe, então o número de classes se igualará ao número de valores inseridos. As áreas não serão criadas para quaisquer locais com valores previstos acima do maior valor de quebra inserido. Você deve inserir pelo menos dois valores e não mais do que 32.

Recortar saída para

Desenhe ou forneça uma camada especificando a área onde você deseja que o resultado seja desenhado. Por exemplo, se você estiver interpolando medições de poluição do lago, você poderá utilizar o limite do lago neste parâmetro e a saída desenhará somente dentro do limite do lago.

Prever nestes locais

Desenhe ou forneça uma camada especificando locais de ponto para calcular valores de predição. Isto permite a você fazer predições em locais de interesse específicos. Por exemplo, se a camada de análise representar medidas de níveis de poluição, você poderá utilizar este parâmetro para predizer os níveis de poluição de locais com grandes populações de risco, como escolas ou hospitais. Você pode então utilizar esta informações para dar recomendações para oficiais de saúde nestes locais.

Se fornecida, esta ferramenta terá uma camada de ponto de saída das predições nestes locais especificados. O nome da camada de saída será o nome da camada resultante com Pontos anexados.

Nome da camada resultante

Forneça um nome para a camada que será criada em Meu Conteúdo e adicionada no mapa.

Ao utilizar a caixa suspensa Salvar resultado em, é possível especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo.
