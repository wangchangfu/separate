การรวมข้อมูลจุด

เครื่องมือนี้ทำงานกับชั้นข้อมูลประเภทจุด (ชั้นข้อมูลที่ใช้วิเคราะห์) และชั้นข้อมูลเชิงพื้นที่ โดยจะพิจารณาจากข้อมูลเชิงจุดที่อยู่ภายในแต่ละพื้นที่ หลังจากที่ตรวจสอบความสัมพันธ์เชิงพื้นที่ของจุดที่อยู่ในพื้นที่, ค่าทางสถิติของจุดที่อยู่ในพื้นที่จะถูกคำนวณและเก็บไว้ในพื้นที่นั้นๆ ค่าทางสถิติพื้นที่ฐานเป็นการนับจำนวนของจุดที่อยู่ภายในพื้นที่ แต่คุณสามารถใช้ค่าสถิติอื่นได้เช่นกัน

ตัวอย่างเช่น สมมติว่าคุณมีข้อมูลตำแหน่งร้านกาแฟและมีข้อมูลพื้นที่เขต และคุณต้องการสรุปจำนวนยอดขายในเขตนั้น สมมติว่าร้านกาแฟมี TOTAL_SALES ข้อมูลเชิงคุณลักษณะ คุณสามารถคำนวณผลลัพธ์ของข้อมูลทั้งหมด TOTAL_SALES ภายในแต่ละพื้นที่ หรือค่าน้อยที่สุดหรือค่ามากที่สุด TOTAL_SALES ภายในแต่ละพื้นที่ หรือค่าเบี่ยงเบนมาตรฐานของยอดขายทั้งหมดที่อยู่ภายในแต่ละพื้นที่

ผลลัพธ์ คือ แผนที่ที่แสดงสีของแต่ละเขต ซึ่งขึ้นอยู่กับจำนวนของตำแหน่งร้านกาแฟต่อขนาดของพื้นที่ การแบ่งการคำนวณโดยใช้พื้นที่เป็นมาตรฐานที่รู้จักกันทั่วไป และยังเป็นเทคนิคในการทำแผนที่ซึ่งใช้ในการแสดงผลข้อมูลเชิงพื้นที่ การทำให้เป็นมาตรฐานช่วยให้คุณเข้าใจแบบรูปของพื้นที่มากขึ้น เนื่องจากขนาดของพื้นที่ถูกนำไปใช้ในการอธิบายเมื่อมีการเปรียบเทียบพื้นที่แต่ละแห่ง

ถ้าเลือก ใช้ขอบเขตแผนที่ปัจจุบัน ใช้ขอบเขตของแผนที่ปัจจุบันในการตรวจสอบ ชั้นข้อมูลและพื้นที่ที่แสดงในหน้าปัจจุบันเท่านั้นที่จะทำการวิเคราะห์ ถ้าไม่เลือก ฟีเจอร์ทั้งหมดที่อยู่ในชั้นข้อมูลวิเคราะห์และชั้นข้อมูลพื้นที่จะถูกนำไปวิเคราะห์ แม้ว่าข้อมูลนั้นจะอยู่นอกขอบเขตแผนที่ที่แสดงอยู่ในปัจจุบันก็ตาม

เลือกพื้นที่

เลือกชั้นข้อมูลพื้นที่จากเว็บแมพที่ข้อมูลประเภทจุดจะอยู่รวมกัน ถ้าไม่มีขอบเขตของชั้นข้อมูลในเว็บแมพ คุณสามารถเลือกจากรายการขอบเขตของชั้นข้อมูลที่มีให้ใน ArcGIS Online ข้อมูลประเภทจุดใดๆ ที่อยู่ภายในขอบเขตของพื้นที่ในชั้นข้อมูลนี้ จะถูกนับและสรุปผลโดยใช้วิธีการทางสถิติที่คุณเลือก

รักษาพื้นที่ไว้ ไม่ให้มีจุด

กำหนดให้ พื้นที่ที่ไม่มีข้อมูลจุด แสดงในชั้นข้อมูลผลลัพธ์ ถ้าไม่เลือก, พื้นที่ที่ไม่มีข้อมูลจุดจะไม่ปรากฎในผลลัพธ์ ถ้าไม่เลือก, พื้นที่ที่ไม่มีข้อมูลจุดจะปรากฎในผลลัพธ์

เพิ่มข้อมูลสถิติ

คุณสามารถคำนวณค่าทางสถิติในตารางคุณลักษณะของชั้นข้อมูลจุด เช่น ค่ารวม, ค่าเฉลี่ย, ค่าน้อยที่สุด, ค่ามากที่สุด และค่าความเบี่ยงเบนมาตรฐาน ชั้นข้อมูลผลลัพธ์จะประกอบด้วยค่าข้อมูลคุณลักษณะใหม่หลังจากการคำนวณทางสถิติ ค่าทางสถิติใดๆ จะสามารถเพิ่มขึ้นโดยเลือกจากข้อมูลคุณลักษณะและค่าสถิติ (กล่องรายการตัวเลือกแบบดึงลงใหม่จะปรากฎขั้น)

เลือกฟีลด์ที่ต้องการให้รวมกลุ่มกัน

คุณสามารถสร้างกลุ่มทางสถิติโดยใช้ตารางคุณลักษณะในชั้นข้อมูลสถิติ ตัวอย่างเช่น ถ้าคุณต้องการรวบรวมเหตุอาชญากรรมที่เกิดขึ้นในพื้นที่รอบๆ คุณต้องใช้ข้อมูลเชิงลักษณะ Crime_type ที่มีข้อมูลอาชญากรรมที่แตกต่างกัน 5 ประเภท โดยอาชญากรรมแต่ละประเภทรวมเป็นหนึ่งกลุ่ม และค่าสถิติที่คุณเลือกจะถูกนำไปคำนวณสำหรับค่าแต่ละค่าที่แตกต่างกันของ Crime_type. ค่าสถิติของแต่ละกลุ่ม เช่นเดียวกับจำนวนนับของข้อมูลจุด จะนำไปแสดงใน pop-up ของชั้นข้อมูลผลลัพธ์ เมื่อคุณเลือกข้อมูลเชิงบรรยายที่ถูกจัดกลุ่ม ผลลัพธ์จะถูกสร้างขึ้น 2 แบบ: ชั้นข้อมูลผลลัพธ์และตารางประกอบด้วยค่าสถิติของแต่ละกลุ่มที่ไม่ซ้ำกัน

ชื่อชั้นข้อมูลผลลัพธ์

ชื่อนี้เป็นชื่อของชั้นข้อมูลที่จะถูกสร้างอยู่ใน เนื้อหาของฉันและถูกเพิ่มเข้าไปบนแผนที่ ชื่อตั้งต้นจะอ้างอิงจากชื่อชั้นข้อมูลที่จะวิเคราะห์ หากชั้นข้อมูลนั้นมีอยู่แล้ว คุณจะถูกถามเพื่อให้ยึนยัน หากคุณต้องการจัดเก็บทับไฟล์เก่า

โดยใช้ บันทึกผลลัพธ์ใน กล่องรายการตัวเลือกแบบดึงลง คุณสามารถกำหนดชื่อของโฟลเดอร์ใน เนื้อหาของฉัน ซึ่งเป็นที่จัดเก็บผลลัพธ์
