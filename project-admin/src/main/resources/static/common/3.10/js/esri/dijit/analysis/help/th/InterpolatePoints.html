การเพิ่มข้อมูลจุด

เครื่องมือเพิ่มข้อมูลจุดนี้ อนุญาตให้คุณคาดการณ์ค่าตัวแปรจากตำแหน่งใหม่ๆ โดยอ้างอิงจากระยะที่วัดได้จากชุดข้อมูลจุดที่สะสมไว้ โดยเครื่องมือนี้จะนำเอาข้อมูลจุดที่มีค่าตัวแปรแต่ละค่า และแก้ไขพื้นที่ให้เป็นไปตามกลุ่มที่มีการคาดการณ์ค่าตัวแปรไว้ ตัวอย่างเช่น

  • ภายในพื้นที่ที่มีการจัดการด้านคุณภาพอากาศ จะมีตัวจับสัญญาณที่ทำหน้าที่วัดระดับมลพิษ การเพิ่มข้อมูลจุด สามารถนำไปใช้คาดการณ์ระดับมลพิษในตำแหน่งต่างๆ ที่ไม่มีเครื่องจับสัญญาณนี้ได้ อย่างเช่น ตำแหน่งที่มีความเสี่ยงต่อประชากร ทั้งในส่วนของโรงเรียนหรือโรงพยาบาล
  • การคาดการณ์ความเข้มข้นของโลหะหนักที่ปะปนอยู่ในพืชผักที่ปลูกไว้ โดยอ้างอิงจากตัวอย่างที่เก็บได้จากพืชแต่ละต้น
  • คาดการณ์ระดับสารอาหารของดิน (ไนโตรเจน ฟอสฟอรัส โพแทสเซียม และอื่นๆ) และตัวชี้วัตอื่นๆ (เช่น การนำไฟฟ้า) เพื่อที่จะศึกษาความสัมพันธ์ระหว่างการเพาะปลูกพืชและปริมาณการให้สารอาหารแต่พืชในแต่ละสถานที่
  • ในทางอุตุนิยมวิทยา ยังรวมไปถึงการคาดการณ์สภาพอากาศ อุณหภูมิ ปริมาณน้ำฝน และตัวแปรอื่นๆ ที่เกี่ยวข้อง (เช่น การเกิดฝนกรด)

เลือกฟิลด์ที่จะเพิ่มเติมข้อมูล

เลือกฟิลด์ที่ประกอบไปด้วยค่าตัวแปรที่คุณต้องการที่จะนำไปเพิ่มเติม

เพิ่มประสิทธิภาพสำหรับ

เลือกตั้งค่าระดับความเร็วและความถูกต้อง ยิ่งต้องการความถูกต้องมากขึ้น จะยิ่งใช้เวลาในการคำนวณมากขึ้น

การคาดการณ์ผลลัพธ์ผิดพลาด

หากเลือก ข้อผิดพลาดมาตรฐานของชั้นข้อมูลในการคาดการณ์การเพิ่มเติมข้อมูลจะถูกสร้างขึ้น และชื่อของชั้นข้อมูลผลลัพธ์จะเป็นชื่อชั้นข้อมูลผลลัพธ์ที่ต่อท้ายด้วยคำว่า ผิดพลาด

ข้อผิดพลาดมาตรฐานนั้นก็มีประโยชน์ เนื่องจากเป็นการแสดงข้อมูลเกี่ยวกับความน่าเชื่อถือของค่าที่คาดการณ์ไว้ กฏพื้นฐานนี้ คือ ค่าที่แท้จริงจะตกอยู่ในช่วงของค่าผิดพลาดมาตรฐานสองค่า โดยคาดการณ์ได้ 95 เปอร์เซ็นต์ของเวลาทั้งหมด ตัวอย่างเช่น การคาดคะเนตำแหน่งใหม่ ต้องใช้ค่าการคาดการณ์ถึง 50 พร้อมกับค่าผิดพลาดมาตรฐานระดับ 5 นั่นความหมายว่าเครื่องมือนี้ได้คาดการณ์ค่าแท้จริงที่ดีที่สุดของตำแหน่งนี้ที่ 50 แต่อาจจะต่ำกว่า 40 ก็เป็นไปได้ หรืออาจจะสูงถึง 60 ก็ได้ ในการคำนวณช่วงค่าอย่างสมเหตุสมผล คูณด้วยค่าผิดพลาดมาตรฐานด้วย 2 แล้วเพิ่มค่าตัวแปรนี้ไปยังค่าที่คาดการณ์เพื่อหาค่าสูงสุดที่อยู่ในช่วงนี้ และหักออกจากค่าที่คาดการณ์ไว้ เพื่อหาค่าต่ำสุดที่อยู่ในช่วงนี้

แบ่งกลุ่มโดยใช้

เลือกค่าที่คาดการณ์ว่าจะถูกจัดกลุ่มตามพื้นที่ได้อย่างไร

  • Geometrical interval - การแบ่งกลุ่มออกตามจำนวนช่วงนั้นของชุดข้อมูลทางเรขาคณิต ด้วยวิธีนี้จะมั่นใจได้ว่า แต่ละช่วงชั้นจะมีระยะห่างที่ใกล้เคียงกันและมีการเปลี่ยนแปลงระหว่างช่วงชั้นอย่างสม่ำเสมอ
  • Equal interval - พื้นที่ถูกสร้างขึ้นจากช่วงของค่าที่คาดการณ์ไว้เท่าๆ กัน ในแต่ละพื้นที่
  • Equal area - พื้นที่ที่ถูกสร้างขึ้นจากจำนวนของข้อมูลในแต่ละพื้นที่ โดยมีขนาดเท่าๆ กัน ตัวอย่างเช่น หากข้อมูลมีค่าขนาดใหญ่ มากกว่าค่าขนาดเล็ก พื้นที่ส่วนใหญ่จะถูกสร้างขึ้นสำหรับค่าขนาดใหญ่
  • Manual - ทางเลือกนี้อนุญาตให้คุณกำหนดช่วงของค่าสำหรับพื้นที่ได้ด้วยคุณเอง ค่าต่างๆ เหล่านี้ จะถูกรวบรวมอยู่ใน ค่าการแบ่งช่วงชั้น ตามพารามิเตอร์ด้านล่าง

จำนวนช่วงชั้น

กำหนดจำนวนช่วงชั้น (ช่วงค่าที่คาดการณ์) ในชั้นข้อมูลผลลัพธ์ แต่ละช่วงชั้นจะกำหนดขอบเขตของพื้นที่ โดยค่าของช่วงชั้นเหล่านี้จะถูกกำหนดขึ้นจาก แบ่งกลุ่มโดยใช้ ตามค่าพารามิเตอร์ด้านบน

ค่าการแบ่งช่วงชั้น

ใส่จำนวนช่วงชั้นที่ต้องการ และใช้การเว้นวรรคเป็นตัวแบ่ง ตัวอย่างเช่น ถ้าคุณต้องการแบ่งช่วงชั้นที่ค่า 1.5, 3, และ 7 ให้ใส่ตัวเลข 1.5 3 7 ค่าเหล่านี้จะใช้ในการกำหนดค่าสูงสุดของแต่ละช่วงชั้น ฉะนั้นจำนวนของช่วงชั้นจะต้องเท่ากับจำนวนค่าที่ใส่ลงไปนี้ อย่างไรก็ตาม พื้นที่จะไม่สามารถสร้างขึ้นในตำแหน่งใดๆ ได้ หากมีการใส่ค่าที่มากกว่าค่าการแบ่งช่วงชั้น โดยคุณต้องใส่อย่างน้อยสองค่า แต่ไม่เกิน 32 ค่า

ตัดผลลัพธ์ออกไปยัง

ไม่ว่าจะเป็นการวาด หรือการจัดเตรียมชั้นข้อมูลที่ระบุพื้นที่ที่คุณต้องการให้ผลลัพธ์แสดงออกมา ตัวอย่างเช่น ถ้าคุณกำลังเพิ่มข้อมูลระดับมลพิษของทะเลสาบ คุณสามารถใช้ขอบเขตของทะเลสาบเป็นค่าพารามิเตอร์ได้ โดยผลลัพธ์ที่ได้จะแสดงเฉพาะบริเวณที่อยู่ในขอบเขตของทะเลสาบเท่านั้น

คาดการณ์ที่ตำแหน่งเหล่านี้

ไม่ว่าจะเป็นการวาด หรือการจัดเตรียมชั้นข้อมูลที่ระบุตำแหน่งในการคำนวณหาค่าที่คาดการณ์ไว้ โดยจะอนุญาตให้คุณสร้างการคาดการณ์ไว้ที่ตำแหน่งที่สนใจที่เลือกไว้ ตัวอย่างเช่น ถ้าชั้นข้อมูลที่ทำการวิเคราะห์แทนด้วยระดับมลพิษ คุณสามารถใช้ค่าพารามิเตอร์นี้คาดการณ์ระดับมลพิษในบริเวณนี้ได้ พร้อมด้วยความเสี่ยงที่อาจเกิดขึ้นกับประชากร เช่น โรงเรียน หรือโรงพยาบาล รวมทั้ง คุณยังสามารถใช้ข้อมูลเหล่านี้ในการให้คำแนะนำแก่เจ้าหน้าที่สาธารณสุขที่อยู่ในบริเวณเหล่านั้น

ผลลัพธ์ที่ได้จากเครื่องมือนี้ เป็นชั้นข้อมูลจุดที่มีการคาดการณ์ตำแหน่งที่ระบุไว้ ชื่อของชั้นข้อมูลผลลัพธ์ได้มาจากผลลัพธ์ที่ได้ พร้อมด้วยข้อมูลจุดที่เพิ่มเข้าไป

ชื่อชั้นข้อมูลผลลัพธ์

ชื่อชั้นข้อมูลที่ได้จะถูกสร้างขึ้นอยู่ใน เนื้อหาของฉัน และถูกเพิ่มเข้าไปบนแผนที่

การใช้ บันทึกผลลัพธ์ลงใน เลือกกล่องรายการตัวเลือกแบบดึงลง คุณสามารถกำหนดชื่อของโฟลเดอร์ใน รายการของฉัน ซึ่งเป็นที่จัดเก็บผลลัพธ์
