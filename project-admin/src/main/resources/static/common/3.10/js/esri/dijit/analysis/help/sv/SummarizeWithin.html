Summera inom

Det här verktyget används för att hitta geoobjekt (och delar av geoobjekt) som ligger inom gränserna för områden i analyslagret. Exempel:

  • Med ett lager för avrinningsgränser och ett lager med markanvändningsgränser enligt markanvändningstyp kan du beräkna den totala arealen för de olika markanvändningstyperna för varje avrinningsområde.
  • Med ett lager för fastigheter i ett län och ett lager kommungränser kan du summera snittvärdet för de tomma fastigheterna inom varje kommungräns.

Om Använd aktuell kartutbredning markeras, analyseras bara de geoobjekt i analyslagret och lagret som ska summeras som visas i den aktuella kartutbredningen. Om alternativet inte är markerat analyseras alla geoobjekt i både indatalagret och lagret som ska summeras, även om de ligger utanför den aktuella kartutbredningen.

Välj lager som ska summeras

Geoobjekt i det här lagret som hamnar innanför gränserna för geoobjekt i analyslagret summeras.

Lägg till statistik från

Alternativt kan du beräkna statistik för numeriska attribut hos punkter som hamnar inom geoobjektens gränser.

Om du markerar kryssrutan Antal punkter beräknas det totala antalet punkter inom varje områdesgräns. Dessutom kan du beräkna statistik för numeriska attribut hos de punkter som ligger inom varje områdesgräns.

  • Summa – Totalvärdet för det angivna fältet.
  • Genomsnitt – Snittvärde eller medelvärde för det angivna fältet.
  • Minimum – Det lägsta värdet för det angivna fältet.
  • Maximum – Det högsta värdet för det angivna fältet.
  • Std. Stdavvikelse – Standardavvikelsen för värdena i det angivna fältet.

Välj fält att gruppera efter

Det här är ett attribut för funktionen Summera geoobjekt som du kan använda för att beräkna statistik separat för varje unikt attributvärde. Anta till exempel att analyslagret innehåller kommungränser och att geoobjekten som ska användas med Summera är fastigheter. Ett av attributen för fastigheterna är Status, som innehåller två värden: LEDIG och UPPTAGEN. Om du vill beräkna den totala ytan för lediga och upptagna fastigheter inom kommungränserna använder du Status som attribut för grupperingen. Statistiken för varje grupp visas i resultatlagrets popupfönster tillsammans med antalet geoobjekt inom varje områdesgräns.

Resultatlagrets namn

Det här är namnet på lagret som skapas i Mitt innehåll och läggs till i kartan. Standardnamnet bygger på analyslagrets namn. Om lagret redan finns uppmanas du bekräfta att du vill skriva över det.

Om du använder listrutan Spara resultat i kan du ange namnet på en mapp i Mitt innehåll där resultatet ska sparas.
