สรุปผลรวมภายใน

เครื่องมือนี้ค้นหาฟีเจอร์ (และส่วนของฟีเจอร์) ที่อยู่ภายในขอบเขตของพื้นที่ในชั้นข้อมูลวิเคราะห์ ตัวอย่างเช่น:

  • มีชั้นข้อมูลขอบเขตลุ่มน้ำและชั้นข้อมูลขอบเขตการใช้ที่ดินแบ่งตามประเภทการใช้ที่ดิน คำนวณหาพื้นที่ทั้งหมดของการใช้ที่ดินแต่ละประเภทที่อยู่ในแต่ละพื้นที่ลุ่มน้ำ
  • มีชั้นข้อมูลที่ดินของเขตการปกครองหนึ่งและชั้นข้อมูลขอบเขตเมือง สรุปพื้นที่ของที่ดินว่างทั้งหมดที่อยู่ในแต่ละเมือง

หาก ใช้ขอบแขตแผนที่ปัจจุบันถูกเลือก จะมีเพียงฟีเจอร์ที่อยู่ภายในชั้นข้อมูลที่ต้องการสรุปที่มองเห็นภายในขอบเขตแผนที่ปัจจุบันเท่านั้นที่ถูกนำมาวิเคราะห์ หากไม่ถูกเลือก ทุกๆ ฟีเจอร์ในทั้งสองชั้นข้อมูลจะถูกวิเคราะห์ แม้ว่าฟีเจอร์นั้นจะอยู่นอกขอบเขตแผนที่ปัจจุบัน

เลือกชั้นข้อมูลผลรวม

ฟีเจอร์ของชั้นข้อมูลนี้ที่ตกอยู่ภายในขอบเขตของฟีเจอร์ในชั้นข้อมูลวิเคราะห์จะถูกนำมาสรุป

เพิ่มการคำนวณทางสถิติจาก

คุณสามารถคำนวณทางสถิติได้จากข้อมูลคุณลักษณะของข้อมูลจุดที่ตกอยู่ภายในขอบเขตของฟีเจอร์

หากเลือก จำนวนจุดที่กล่องตัวเลือก ข้อมูลจะถูกนำมาคำนวณหาจำนวนจุดทั้งหมดภายในแต่ละขอบเขต นอกจากนั้น คุณยังสามารถคำนวณทางสถิติจากข้อมูลคุณลักษณะของข้อมูลจุดที่อยู่ภายในแต่ละขอบเขตได้ด้วย

  • รวม—ค่ารวมของฟีลด์ที่เลือก
  • เฉลี่ย—ค่าเฉลี่ยของฟีลด์ที่เลือก
  • ค่าต่ำสุด—ค่าต่ำสุดที่อยู่ในฟีลด์ที่เลือก
  • ค่าสูงสุด—ค่าสูงสุดที่อยู่ในฟีลด์ที่เลือก
  • ค่ามาตรฐาน เบี่ยงเบนมาตรฐาน—ค่าเบี่ยงเบนมาตรฐานของค่าทั้งหมดที่อยู่ในฟีลด์ที่เลือก

เพิ่มการคำนวณทางสถิติจาก

คุณสามารถคำนวณทางสถิติได้จากข้อมูลคุณลักษณะของฟีเจอร์เส้นที่อยู่ใกล้เคียง

กล่อง ความยาวของเส้นหากถูกเลือก จะคำนวณหาความยาวของฟีเจอร์เส้นทั้งหมดที่อยู่ภายในแต่ละขอบเขต นอกจากนั้น คุณยังสามารถคำนวณทางสถิติจากข้อมูลคุณลักษณะของข้อมูลเส้นที่อยู่ภายในแต่ละขอบเขตได้ด้วย

  • รวม—ค่ารวมของฟีลด์ที่เลือก
  • เฉลี่ย—ค่าเฉลี่ยของฟีลด์ที่เลือก
  • ค่าต่ำสุด—ค่าต่ำสุดที่อยู่ในฟีลด์ที่เลือก
  • ค่าสูงสุด—ค่าสูงสุดที่อยู่ในฟีลด์ที่เลือก
  • ค่ามาตรฐาน เบี่ยงเบนมาตรฐาน—ค่าเบี่ยงเบนมาตรฐานของค่าทั้งหมดที่อยู่ในฟีลด์ที่เลือก

เพิ่มการคำนวณทางสถิติจาก

คุณสามารถคำนวณค่าทางสถิติได้จากข้อมูลคุณลักษณะของฟีเจอร์พื้นที่ที่อยู่ใกล้เคียง

กล่อง สรุปขนาดพื้นที่ที่ใช้หากถูกเลือก จะคำนวณพื้นที่ทั้งหมดของฟีเจอร์ที่อยู่ภายในขอบเขตแต่ละขอบเขตของชั้นข้อมูลวิเคราะห์ นอกจากนั้น คุณยังสามารถคำนวณทางสถิติจากข้อมูลเชิงบรรยายของฟีเจอร์ที่อยู่ภายในแต่ละขอบเขตได้ด้วย

  • รวม—ค่ารวมของฟีลด์ที่เลือก
  • เฉลี่ย—ค่าเฉลี่ยของฟีลด์ที่เลือก
  • ค่าต่ำสุด—ค่าต่ำสุดที่อยู่ในฟีลด์ที่เลือก
  • ค่าสูงสุด—ค่าสูงสุดที่อยู่ในฟีลด์ที่เลือก
  • ค่ามาตรฐาน เบี่ยงเบนมาตรฐาน—ค่าเบี่ยงเบนมาตรฐานของค่าทั้งหมดที่อยู่ในฟีลด์ที่เลือก

เลือกฟีลด์ที่ต้องการให้รวมกลุ่มกัน

นี่คือข้อมูลคุณลักษณะของ การสรุปฟีเจอร์ที่คุณสามารถใช้คำนวณทางสถิติจากค่าในข้อมูลคุณลักษณะ ตัวอย่างเช่น ชั้นข้อมูลวิเคราะห์มีข้อมูลขอบเขตเมืองและฟีเจอร์ที่ต้องการ สรุปเป็นที่ดิน ตารางเชิงคุณลักษณะหนึ่งของแปลงที่ดินคือ Status ประกอบด้วยค่า 2 ค่า: ว่าง และ ไม่ว่าง ในการคำนวณพื้นที่ทั้งหมดของแปลงที่ดินที่ว่างและไม่ว่างภายในขอบเขตของเมือง ให้ใช้ Status ตามกลุ่มของตารางเชิงคุณลักษณะ ค่าสถิติของแต่ละกลุ่ม เช่นเดียวกับจำนวนฟีเจอร์ที่อยู่ภายในขอบเขตแต่ละพื้นที่ จะถูกนำมาแสดงบน pop-up ของชั้นข้อมูลผลลัพธ์

ชื่อชั้นข้อมูลผลลัพธ์

ชื่อนี้เป็นชื่อของชั้นข้อมูลที่จะถูกสร้างอยู่ใน เนื้อหาของฉันและถูกเพิ่มเข้าไปบนแผนที่ ชื่อตั้งต้นเชื่อมโยงกับชื่อชั้นข้อมูลที่ใช้วิเคราะห์ หากชั้นข้อมูลนั้นมีอยู่แล้ว คุณจะถูกถามเพื่อให้ยึนยัน หากคุณต้องการจัดเก็บทับไฟล์เก่า

โดยใช้ บันทึกผลลัพธ์ใน กล่องรายการตัวเลือกแบบดึงลง คุณสามารถกำหนดชื่อของโฟลเดอร์ใน เนื้อหาของฉัน ซึ่งเป็นที่จัดเก็บผลลัพธ์
